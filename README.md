# PDE (Personal Decision Engine)

> Helping people make better decisions by understanding who they are.

## What is this repository?

This repository contains the complete strategy, product vision and technical documentation for the PDE project.

PDE is an attempt to build a new kind of consumer platform.

Instead of creating another isolated application (movies, restaurants, books, music...), the goal is to understand the user across all of those domains and help them make better decisions using that knowledge.

We believe the future is not another recommendation engine.

The future is a **Personal Decision Engine**.

## Repository philosophy

This repository is not intended to document software.

It is intended to document thinking.

Every important decision should answer one of these questions:

- What do we believe?
- Why do we believe it?
- How are we going to validate it?
- What have we learned?
- What should we build next?

Code is expected to change.

The vision should evolve much more slowly.

Exploratory work should happen locally first. GitHub should receive coherent versions worth preserving, not every intermediate draft.
